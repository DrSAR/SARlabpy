sarpy Package¶

Central __init__.py file that pulls various typically used functions from the various subpackes into the main sarpy namespace. This way, e.g., instead of using the cumbersome sarpy.io.BRUKER_classes.Scan() users can use the rather snappy sarpy.Scan().
